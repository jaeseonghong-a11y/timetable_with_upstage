## 시작 순서 (처음 한 번)

### 1단계 — 폴더 준비

1. 이 킷의 **모든 파일을 새 프로젝트 폴더**에 넣는다 (루트에).
2. VS Code로 그 폴더를 연다.

### 2단계 — Git·GitHub

```bash
git init
git add -A
git commit -m "chore: 프로젝트 하네스 초기 세팅"
# GitHub에서 리포 생성 후:
git remote add origin [내-리포-주소]
git push -u origin main
```

### 3단계 — 두 도구 준비

- VS Code 터미널에서 **Claude Code**와 **Codex**를 각각 설치·로그인.
- 터미널 2개를 열어 한쪽=Claude Code, 한쪽=Codex로 쓰면 편하다.

### 4단계 — 품질 자동화 켜기 (권장)

```bash
pip install pre-commit
pre-commit install
```

→ 이제 커밋할 때마다 린터·키검사가 자동으로 돈다.

### 5단계 — 첫 개발 세션 시작

Claude Code 또는 Codex 터미널에 **PROMPTS.md의 "1. 아주 처음" 프롬프트**를 복붙한다.
그러면 AI가 문서를 읽고 개발 계획을 제안한다. 승인하면 코딩 시작.

## 딱 3가지만 기억하면 됨

1. **맥락은 대화가 아니라 파일로 이어진다.** (CURRENT_STATE.md가 다리)
2. **작업 멈출 땐 CURRENT_STATE.md를 갱신**하고 멈춘다.
3. **새 도구로 넘어가면 AGENTS.md + CURRENT_STATE.md부터 읽힌다.**

이 3개만 지키면 토큰이 떨어져도, 도구를 바꿔도, 며칠 뒤에 와도 맥락이 안 끊긴다.
