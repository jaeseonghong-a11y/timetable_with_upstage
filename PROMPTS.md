## 🟣 5. 교차 리뷰 — 한 도구가 짠 걸 다른 도구가 검토

```
방금 커밋된 변경(git diff HEAD~1)을 리뷰해줘. 다음 관점으로:
- AGENTS.md 규칙 위반이 없는가
- API 키 노출·보안 문제가 없는가
- Upstage가 서비스 심장 역할을 유지하는가 (크롤링이 비대해지지 않았나)
- 죽은 코드·중복 로직이 생기지 않았나
문제를 목록으로 먼저 보고하고, 고칠지는 내가 정할게.
```
