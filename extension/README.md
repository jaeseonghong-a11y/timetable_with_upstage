# SKKU Timetable Connector

`SKKU TIMETABLE` 웹서비스에서 선택한 과목을 에브리타임 강의평으로 연결하는 개인용 브라우저
확장프로그램이다. Chrome·Edge·Whale의 Chromium 확장프로그램 형식을 사용한다.

## 하는 일

- 교수명이 있으면 교수명으로 먼저, 비어 있으면 과목명으로 에타 강의평 검색 화면을 연다.
- 검색 결과 중 과목명과 교수명이 하나로 일치할 때만 그 `lecture/view/{id}` 주소로 이동한다.
- 성공한 **주소만** 이 브라우저의 확장프로그램 저장소에 보관한다. 이후에는 바로 강의평을 연다.
- 동일한 과목·교수 결과가 여러 개면 자동으로 추측하지 않고, 에타 탭에서 사용자가 한 번 고르게
  한다. 선택된 주소만 기억한다.
- `담은 과목 강의평 자동 연결`은 최대 12개 선택 과목만 순차 처리하며, 이미 기억한 항목은 즉시
  건너뛴다.

## 하지 않는 일

- 강의평 본문, 별점, 댓글, 작성자, 로그인 쿠키를 읽거나 서버로 보내지 않는다.
- 에타의 비공개 API를 호출하지 않는다.
- 전 과목을 사전 수집·미러링하지 않는다.
- GLS 책가방 추가나 에타 시간표 쓰기는 이 버전에 포함하지 않는다. 인증 뒤 시스템을 자동 조작하는
  기능은 서비스 정책·안전성 검토 후 별도 동의를 받아야 한다.

## 설치(개발·데모용)

1. Chrome·Edge·Whale 주소창에 `chrome://extensions`를 연다.
2. 우측 상단의 **개발자 모드**를 켠다.
3. **압축해제된 확장 프로그램을 로드합니다**를 누른다.
4. 이 `extension/` 폴더를 선택한다.
5. 에브리타임에 로그인하고, `https://timetable-with-upstage.vercel.app`를 새로고침한다.

로컬 개발 시에는 `http://localhost:3000` 또는 `http://localhost:3001`도 이미 허용되어 있다.
코드를 고친 뒤에는 확장프로그램 관리 화면에서 새로고침 아이콘을 누른다.

## 사용

- 과목 목록의 분반 또는 생성된 시간표 카드에서 **에타 강의평 보기**를 누른다.
- 설치되어 있으면 에타 탭에서 자동으로 과목·교수명을 대조한다.
- 처음부터 정확한 하나를 찾으면 바로 강의평으로 이동한다.
- 동명이거나 교수 정보가 부족하면 강조된 에타 결과 중 하나를 고른다. 다음부터는 즉시 연결된다.

## 검증

```powershell
cd extension
npm test
```

매칭 규칙은 과목명과 교수명 모두가 일치할 때만 자동 이동하도록 테스트한다.
