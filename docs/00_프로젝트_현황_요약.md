■ 현재 메인 (확정 — 옵션 A 채택, D-12)
"강의계획서를 읽어 학기 부담까지 예측하는 시간표 조합 추천 서비스" (성균관대 학부생 대상)
- 개설강좌 수집(scraper)은 최소 시딩. 서비스 심장은 강의계획서 PDF를 Upstage Document Parse/Extract로
  구조화해 평가방식·과제비중·시험일정·수업유형을 뽑아내는 것.
- 시간 제약 + Upstage파생 부담도(과제비중·시험집중도·수업형태) 2축으로 조합을 정량화해 상위 N개 역제안.
- 각 조합의 추천 사유를 Solar가 생성 ("이 조합은 과제가 몰려있다" 등).
- 기존 시간표앱(에타)이 "충돌 체크"만 하는 것과 반대로, "조건 넣으면 부담도까지 고려해 순위매긴 조합 추천".
