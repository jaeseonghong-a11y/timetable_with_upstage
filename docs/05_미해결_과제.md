- Solar가 실제 캡처의 제1전공 3행만 반환한 경우도 수정했다. 전체 행 추출 지시를 강화하고, 누락은
  Document Parse의 Markdown/HTML 표에서 원래 순서로 보완한다(rowspan 병합 셀 포함).
- 균형교양 세 행은 동일 그룹의 `3개 영역 중 최소 2개 영역, 합계 6학점` 규칙으로 구조화했다.
  한 영역 6학점만으로는 불충족이며, 두 영역 이상에 걸쳐 합계 6학점이면 충족한다.
- 수강/취득과목 원본에서 정상 행이 전부 검토함으로 빠지던 문제를 해결했다. 학수번호 숫자 3~4자리,
  한글 및 열 경계에서 분리된 학기 표기를 정규화하고, Document Parse 표의 코드·과목명 쌍으로 Solar
  누락을 보완한다. 한 셀에 합쳐진 서로 다른 두 과목도 분리하며 원본 실측은 33과목·학기 33개·
  잘못된 행/검토 사유 0건이다.
- 기수강 데이터의 `exclude/retake`는 현재 검토 초안부터 상위 세션을 거쳐 실제 전공 후보에 연결한다.
  최종 확정은 후속 추천 데이터 승인에 계속 사용하되, 과목 선택에서 이미 이수한 과목을 숨기는 데는
  별도 확정을 요구하지 않는다. 실제 `ADD2003`·`ADD2021` 회귀 테스트를 추가했다.
