■ 메인 후보
═══════════════════════════════════════

【① 시간표 조합 추천】 ★ 현재 메인
- 요약: 과목 후보 넣으면 시간·평점·교수·과목성격 정량화해 최적 조합 N개 역제안 + Solar 추천사유.
- novelty 중~강 / value 중 / 대회적합성 강 / 도메인방어 중
- 강점: 5개 심사축 모두 충족. 데이터 확보 검증 완료.
- ★ 약점(미해결): 현 구도는 크롤링+조합계산이 심장, Upstage가 곁가지 → 필연성 취약.
  해결책 = 옵션A(강의계획서 파싱 중심 재설계). 01_의사결정로그 D-10 참조.
- 리스크: 경우의수 폭발(필터·상위N 필수) / 평점 데이터소스 미확정 / "정량지표 뭘로?" 정의 필요.

■ 대안 후보 (메인 엎어지면)
═══════════════════════════════════════

【매치유 — 교환학생 매칭 (재설계판)】 2순위
- 요약: 협정대학 엑셀 + 내 조건(전공·어학·예산·성향) → 최적 학교 추천·비교.
- novelty 강 / value 중 / 대회적합성 중~강
- 강점: 3축 융합(전공×예산×성향) = 심사 상단 기대치. 문서 업로드 기반이라 뿌리 맞음.
- 조건: 원안은 웹검색 의존 → "요람·물가표 PDF 업로드+Parse"로 무게중심 옮겨야 함. Claude API→Solar.
- 리스크: 커리큘럼·물가를 문서로 확보 시 사용자부담↑ (트레이드오프).

【D9 — 규제문서 × Cesium 공간정착】 2순위
- 요약: 규제·인허가·민원 문서를 Parse/Extract → 3D지도(Cesium) 위 위치에 핀. "건물 클릭→규제 요약".
- novelty 최강 / value 약(스코프) / 대회적합성 강 / 도메인방어 강(재성 AGE 자산 직결)
- 강점: "문서×공간" 융합은 대회에서 거의 안 나올 접근. 챗봇 차별화 최강. AGE Cesium 코드 재활용.
- 리스크: 7/25 스코프 초과 위험 최대(문서파싱+지오코딩+3D통합). 필지 지오코딩 난이도. 규제문서 확보.

【D10 — 워런티/리콜 추적】 (안전패, novelty 약)
- 요약: 영수증·보증서 사진 → 제품별 보증만료·리콜 자동정리·알림.
- novelty 중 / value 강 / 대회적합성 중~강
- 강점: 영수증은 Extract가 잘 먹음(정형). 시연 쉬움.
- 약점: novelty 낮음(기존 영수증앱 존재). 재성 관심사와 거리. 리콜 데이터원 불확실(빠지면 그냥 정리앱).

【② 3D 프린팅 설정 KB】 (재성 강점, 대회적합성 약)
- 요약: 필라멘트 데이터시트 PDF + 실패로그 → 소재별 최적 슬라이서 설정 추천·실패원인 매칭.
- novelty 중 / value 중 / 대회적합성 약 / 도메인방어 강(재성 Bambu P1S 실무)
- 강점: 본인 실무라 "왜 내 서비스인가" 진정성 최강. 데이터시트는 공개PDF라 확보 쉬움.
- 약점: "흩어진 문서"가 애매(설정지식은 문서보다 경험지). 설정값 틀리면 출력실패(정확도 부담 큼).
- 살리려면: "다수 제조사 데이터시트 PDF 융합"으로 문서기반 명확히 + 추천값에 출처·"권장 시작값" 명시.

■ 드롭된 후보 (이유와 함께 보존 — 재부활 방지)
═══════════════════════════════════════

【구직 공고 크롤링 (조원案)】 ✗ 드롭
- 이유: 크롤링이 심장 + 법적리스크(조원 스스로 잡코리아 판례 언급) + Upstage 미사용(Claude API).
- 부분생존: "이력서 vs 공고 갭분석"은 업로드 기반 재설계 시 유효 = "채용공고 갭 분석기".

【리얼프라이스 — 뷰티 최저가 (조원案)】 ✗ 드롭
- 이유: 기획완성도는 최고급이나 "가격비교 서비스지 Document AI 아님". 크롤링·API가 90%.
  Upstage 억지로 끼우면 "세일공지 파싱"만 남아 필연성 방어 불가.
- 권고: 대회 밖 개인 프로젝트로.

【디택 — 직구 원본매칭 (조원案)】 ✗ 드롭
- 이유: "흩어진 문서" 없음(옷 이미지·링크). 비전 역이미지검색이지 문서AI 아님.
  정확도 도박(단추·재봉선 원본특정 어려움) + 실시간 타오바오/쉬인 불안정 + 택갈이 회색지대.
